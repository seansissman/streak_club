# Streak Engine

Streak Engine is a Devvit Web app for running and tracking streak-based gameplay on Reddit.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Authenticate with Devvit (if not already logged in):

```bash
npx devvit login
```

## Deploy and Test

Upload the app:

```bash
npx devvit upload
```

Start playtest:

```bash
npx devvit playtest
```

Daily reset is at **00:00 UTC**.
